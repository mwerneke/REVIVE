## 🎯 What The Tool Does

### Input (What You Type)
- How many skins? (Volume)
- What price? (Selling price)
- Any discount? (%)
- When do they pay? (Payment days)
- Your monthly overhead (optional)

### Output (What You Get)
- 💵 **Your Profit** (Dollar amount)
- 📊 **Margin %** (Profitability percentage)
- 🔴/🟡/🟢 **Risk Level** (Is it risky?)
- ✅/⚠️/❌ **Decision** (Take it or negotiate)

**That's it. No complexity. No confusion.**
